## Introduction
A small Web Storage application using `vue3`, `vite`, `Shadcn UI`, `TypeScript`, `Supabase`.

## Install and use

- Get the project code

```bash
git clone https://github.com/makcmakc/cloud-storage.git
```

- Installation dependencies

```bash
cd cloud-storage

pnpm install
```

- run

```bash
pnpm dev
```

- build

```bash
pnpm build
```
